# Roadmap de evolução

- **v0.1**: Prova de conceito (gerador de IPs, verificador básico de regras).
- **v0.5**: Sniffer que detecta IPs com taxa alta de pacotes e aplica bloqueio (iptables) — POC.
- **v1.0**: Leitura de listas (whitelist/blacklist), logging estruturado, containerização e testes automatizados.
- **v2.0**: Integração com SIEM, painel web para visualização (Grafana/Prometheus), modo de simulação sem necessidade de root.
